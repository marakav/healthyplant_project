# HealthyPlant - Уход за комнатными растениями

Веб-приложение для анализа и ухода за комнатными растениями через Telegram бота.

## Функции
- Регистрация и авторизация пользователей
- Генерация токенов для анализа растений
- Взаимодействие с Telegram ботом @HealthyPlants_Bot
- Сохранение истории анализов
- Рекомендации по уходу за растениями

## Использование
1. Зарегистрируйтесь на сайте
2. Получите токен в личном кабинете
3. Отправьте токен и фото растения в Telegram бота @HealthyPlants_Bot
4. Получите результат анализа на сайте

## Технологии
- HTML5, CSS3, JavaScript
- LocalStorage для хранения данных
- Адаптивный дизайн
